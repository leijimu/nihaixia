倪海厦（1954-2012），生于台湾，早年毕业于东吴大学政治学系。少时曾梦想成为摇滚吉他手，后因阅读《医宗金鉴》自学治愈亲人妇科疾病，遂与中医结下不解之缘。他博古通今，除精通医术外，亦擅长命理、占卜与风水（化名梵宇龙），展现了传统文化通才的造诣。

1990年代初，倪师赴美创立“汉唐经方中医诊所”及汉唐中医学院。他曾于2000年至2003年受命担任美国佛罗里达州针灸委员会委员及副主席，致力推动针灸立法，并获侨委会颁发海外优秀华人奖。倪师一生力排众议，强烈批判西方医学之流弊，誓言复兴《伤寒》、《金匮》之正统经方。晚年更受邀至上海中医药大学扶阳论坛及斯坦福大学（Stanford University）演讲，吸引千人聆听，引起巨大反响。

为将毕生所学传承后世，倪师耗尽心血录制《人纪》教学，涵盖针灸、内经、本草、伤寒、金匮五大篇章，成为当代中医教育的重要瑰宝。
